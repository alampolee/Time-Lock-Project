# Time Lock

**Time Lock** is a web-based app designed to maximize productivity by wrapping time into blocks and naming them. It is the perfect tool for performance-seekers looking for a better Pomodoro Technique.

## Features

- **Time Blocks**: Organize your work into focused chunks.
- **Task Naming**: Give each session a specific name to stay on track.
- **Gamified Progress**: Earn XP and level up as you complete focus sessions.
- **Visual Aesthetics**: A "Deep Void" aesthetic with neon glass UI elements.

## Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher recommended)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)

### Installation

1.  **Clone the repository** (or download the source code):
    ```bash
    git clone <repository-url>
    cd "Time Lock Project"
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

### Running the App

1.  **Start the development server**:
    ```bash
    npm run dev
    ```

2.  **Open your browser** and navigate to the URL provided in the terminal (usually `http://localhost:5173`).

## Technologies Used

- **React**: For the user interface.
- **Vite**: For fast development and building.
- **Three.js & React Three Fiber**: For 3D visual effects.
- **Zustand**: For state management.
- **Framer Motion**: For smooth animations.
